# Demande d'accès

1. Vous pouvez effectuer vos demandes d'accès à Phileas directement via le portail MesoNET : https://acces.mesonet.fr.
2. Une fois votre compte MesoNET validé, et votre accès au projet accordé, connectez vous à https://auth.phileas.ec-nantes.fr
3. __ Après vous être connecté la première fois, veuillez patienter au moins 15 minutes que les projets du portail MesoNET soient synchronisés sur votre utilisateur Phileas__
