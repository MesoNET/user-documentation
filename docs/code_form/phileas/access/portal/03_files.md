# Gestion des fichiers
Sur le portail, vous pourrez gérer vos fichiers directement depuis votre navigateur.

:::info
Cet accès aux fichiers est un accès pratique, mais non prévu pour des transferts longs, ou de gros fichiers. Pour ces utilisations particulières, veuillez suivre [la documentation associée](../ssh_transfers/)

:::

![Menu de gestion des fichiers](/img/phileas/portal_filemenu.png)

Ce menu vous permettra d'accéder à tous vos espaces de stockage disponibles sur Phileas. Vous pourrez également, si nous avons fait la manipulation, accéder à vos données de Nautilus si besoin.

Vous pourrez ainsi télécharger, téléverser, et gérer vos fichiers instantanément.

![Modifier les fichiers](/img/phileas/portal_filegestion.png)

Si vous n'avez besoin de modifier qu'un seul petit fichier texte, un éditeur est directement proposé dans le menu _Edit_.

![Modifier les fichiers](/img/phileas/portal_fileedit.png)
