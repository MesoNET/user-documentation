## Demande de projet Nova

:::warning Attention
Pour avoir accès à un projet Nova, vous devrez procéder comme pour n'importe quelle demande de projet MesoNET et passer par le [portail Mesonet](https://acces.mesonet.fr). Sur le portail, vous pourrez préciser les ressources souhaitées (nombres de VMs, CPU, RAM, Disque). Voir la [section](/acces/portail) dédiée au portail.
:::

## Un problème avec un projet NOVA ?

* Vous pouvez [ouvrir un ticket](https://tickets.mesonet.fr/) ou écrire un mail à [support@mesonet.fr](mailto:support@mesonet.fr).
* Pour plus de détails, voir la section [Support](/support/tickets).
