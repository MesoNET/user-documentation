## OpenStack
[OpenStack](https://www.openstack.org/) est un ensemble de logiciels open source qui offre une solution pour déployer et gérer des infrastructures de cloud computing.


## Nova : plateforme de Cloud Computing de GRICAD
Nova est la plateforme OpenStack de [GRICAD](https://gricad.univ-grenoble-alpes.fr/) qui vous propose un service de machines virtuelles **non pérennes** à la demande.
Accessible via une interface web, un CLI ou une API, la plateforme permet de gérer des VMs, leur réseau et toutes les autres ressources liées au Cloud Computing.
